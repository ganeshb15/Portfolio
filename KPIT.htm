Fixed-Point Arithmetic

Fixed-point arithmetic is used in embedded systems to represent numbers with both integer and fractional parts, but without using floating-point operations. It uses integers with a fixed scaling factor.

Key Points:

  • Q Format: Numbers are represented as Qm.n, where m is the number of integer bits, and n is the number of fractional bits.
  • Precision and Range: More fractional bits increase precision, and more integer bits increase range.
  • Addition/Subtraction: Simple like integer math if numbers have the same scaling.
  • Multiplication: After multiplying two fixed-point numbers, the result needs to be shifted to the right by the scaling factor.
  • Division: The dividend is scaled up before dividing to maintain accuracy.
  • Overflow/Underflow: Special handling like saturation is used to manage exceeding the limits of the number format.

Fixed-point math is faster and uses less memory than floating-point, making it ideal for embedded systems like automotive ECUs.

MC/DC (Modified Condition/Decision Coverage)

MC/DC Coverage is a software testing method used in safety-critical systems like automotive or aerospace software. It ensures that decisions in the code are tested in a way that each condition within the decision is independently evaluated.

Key Points:

  • Decision Testing: Ensures every decision (e.g., an if statement) is tested for both true and false outcomes.
  • Condition Testing: Makes sure that each condition in a decision affects the outcome independently.
  • Test Coverage: Tests enough cases to show how each condition influences the decision, but not every combination is required.

Example:

if (A && B)
{
    // Do something
}
  • Test case 1: A = True, B = True (decision is True)
  • Test case 2: A = False, B = True (changing A makes the decision False)
  • Test case 3: A = True, B = False (changing B makes the decision False)

Summary: MC/DC ensures each condition in a decision is tested for its impact, making it an efficient and thorough testing approach for critical systems.
