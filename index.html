$ BreadboardScript: High-Level Language for Circuit Design:

This project involves Logisim as a digital circuit simulation tool to create a template circuit design specification for high-level languages that can be implemented on breadboard logic. I custom-designed a programming language using Python and the PLY (Python Lex-Yacc) library, crafting my own high-level Arduino-like language. Additionally, I developed a compiler in Python that outputs a custom circuit diagram and the corresponding opcode to implement the final digital circuit on a breadboard.

The project functions similarly to Verilog, but provides higher-level hardware abstractions like RAM, ROM, and registers, designed to be implemented directly on a breadboard. The main objective is to establish a high-level language framework that can be translated into low-level hardware design, facilitating practical hardware development on IC logic.

$ BMP Image Processing in C++:

The BMP Image Processing project involves creating a C++ application that reads a BMP image, processes its pixel data, and generates a new BMP file based on specific conditions. The application is built entirely without using any external libraries or headers, providing a hands-on learning experience in interpreting and utilizing BMP file headers from scratch. It includes a custom header file to manage BMP file structure, read pixel values, and apply a thresholding algorithm that sets pixels above a certain RGB value to white while converting others to black. This approach offers an in-depth understanding of file handling, image processing, and low-level BMP manipulation, serving as a foundation for more advanced image analysis tasks.
